#***************************************************************************************
#-----------------------------------本地创建仓库----------------------------------------
#电脑安装Git后，新创建文件夹一个文件夹，打开文件夹后单击鼠标右键后左击选择Git Bash Here
#--------------------------------------------------------------------------------------
git config --global user.name 'xxxxx'  #------设置用户名，xxxxx输入用户名。
git config --global user.email 'xxxxx'   #----设置邮箱地址，xxxxx处输入邮箱地址。
git init   #----------------------------------初始化数据库。
mkdir xxxxx   #-------------------------------直接创建文件夹
cd xxxxx   #----------------------------------打开文件夹，xxxxx处输入刚刚创建的文件夹名字。
vim xxxx.xxx   #------------------------------创建文件并打开。  
----------------------------------------------
进入编辑界面后英文输入状态下点击键盘A键后可输入
内容，输入完毕后按Shift键+;键后底部可输入字母wq
，然后回车即可保存退出。
----------------------------------------------
git add xxxx.xxx   #--------------------------将工作区文件提交到暂存区。
git commit -m 'xxxxx’   #---------------------将暂存区文件提交到仓库,xxxxx处填写变动说明。
#---------------------------------------------------------------------------------------
#备注1：git config --list   列出所有Git当前找到的配置。
#备注2：git status   显示工作目录和暂存区的状态。
#备注3：touch命令是直接创建文件，vim命令量创建文件并打开编辑。
#备注4：ls -a -l   显示隐藏和非隐藏文件夹和文件、列表方式显示。
#备注5：vim   输入已经存在的文件名即可对之前的文件进行修改。
#备注6：xxxx.xxx代表文件名和后缀名，xxxxx代表文件名。



#****************************************************************************************
#------------------------------------远程克隆仓库----------------------------------------
#电脑安装Git后，新创建文件夹一个文件夹，打开文件夹后单击鼠标右键后左击选择Git Bash Here
#---------------------------------------------------------------------------------------
git config --global user.name 'xxxxx'  #------设置用户名，xxxxx输入用户名。
git config --global user.email 'xxxxx'   #----设置邮箱地址，xxxxx处输入邮箱地址。
git clone xxxxxxxx   #-----------------------克隆远程仓库到本地，xxxxxxxx填写仓库克隆地址。
vim xxxx.xxx   #------------------------------创建文件并打开。  
----------------------------------------------
进入编辑界面后英文输入状态下点击键盘A键后可输入
内容，输入完毕后按Shift键+;键后底部可输入字母wq
，然后回车即可保存退出。
----------------------------------------------
git add xxxx.xxx   #--------------------------将工作区文件提交到暂存区。
git commit -m 'xxxxx’   #---------------------将暂存区文件提交到仓库,xxxxx处填写变动说明。
git push   #----------------------------------将本地变动同步到远程仓库。
#----------------------------------------------------------------------------------------
#备注1：git config --list   列出所有Git当前找到的配置。
#备注2：git status   显示工作目录和暂存区的状态。
#备注3：touch命令是直接创建文件，vim命令量创建文件并打开编辑。
#备注4：ls -a -l   显示隐藏和非隐藏文件夹和文件、列表方式显示。
#备注5：vim   输入已经存在的文件名即可对之前的文件进行修改。
